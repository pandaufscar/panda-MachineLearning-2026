# 🧠 Grupo de Estudo de Machine Learning - Panda UFSCar 2026

## 👥 Sobre o Grupo

O Grupo de Estudo de Machine Learning faz parte dos **Grupos de Estudo 2026** do **Panda UFSCar** (Grupo de Processamento e Análise de Dados).

Nosso objetivo é proporcionar um ambiente colaborativo para aprendizado prático de Machine Learning, partindo do zero e construindo conhecimento de forma gradual e acessível.

Ao longo de **17 semanas**, os participantes terão contato com conceitos fundamentais, bibliotecas essenciais (Python, Pandas, Scikit-learn) e desenvolverão projetos práticos que consolidarão o aprendizado. Todo o material produzido será disponibilizado publicamente neste repositório, contribuindo para a comunidade acadêmica.
